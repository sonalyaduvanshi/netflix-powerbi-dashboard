"Netflix Movies & TV Shows Analysis"

This project analyzes Netflix Movies and TV Shows data using Python for data cleaning and Power BI for building an interactive dashboard.

Tools Used
- Python (Pandas, NumPy)
- Excel
- Power BI

Project Overview
- Cleaned and transformed raw Netflix data
- Analyzed content distribution by type, genre, country, and year
- Built an interactive Power BI dashboard with KPIs and filters

Dashboard Features
- Total titles, movies, and TV shows
- Content growth over the years
- Genre-wise and country-wise analysis
- Interactive filters for year, country, and type
